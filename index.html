Install Reclude for Excel (Windows Beta)

Private beta build for Excel Desktop on Windows.

Download Reclude for Excel for Windows Download manifest (advanced)

Install Steps

  1. Download the Windows installer from this page.
  2. Run the installer and complete the setup flow.
  3. Open Excel Desktop on Windows.
  4. Open Add-ins and launch Reclude.

Beta note: If Excel does not show Reclude immediately after install, fully restart Excel once and reopen your workbook.
